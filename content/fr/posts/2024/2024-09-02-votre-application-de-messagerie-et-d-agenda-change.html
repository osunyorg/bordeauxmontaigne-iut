À compter du 3 septembre, l'application Bureau virtuel utilisée jusqu’à présent pour consulter la messagerie et l'agenda sera remplacée par la nouvelle solution Carbonio.

La rentrée 2024 marquera un changement important concernant la gestion des courriels et agendas personnels des étudiants.

Tous les nouveaux messages arriveront désormais sur la plateforme Carbonio.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Lien d’accès : https://carbonio.u-bordeaux-montaigne.fr/ (lien externe)

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: no_background text: >-

Pour les étudiant·e·s disposant d'un compte Bureau virtuel, celui-ci restera accessible jusqu'au 31 décembre 2024 à l’adresse https://bv.u-bordeaux-montaigne.fr.
Vous pourrez ainsi vous connecter afin de récupérer les éléments que vous souhaitez basculer dans Carbonio.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: accent_background text: >-

Afin de vous guider dans la prise en main du nouvel outil, vous pouvez consulter les fiches d’aide disponibles sur l'espace étudiant : Fiches d'aide pour ma messagerie (lien externe)

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: no_background text: >-

Depuis Carbonio, le transfert vers une boîte mail personnelle n'est pas possible.
