L'IUT dispose de plusieurs bornes d'accès WiFi qui permettent d'accéder à internet depuis un ordinateur personnel ou appareil mobile.

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 68 text: >- 1 minute contents: - kind: block template: title title: >- Le WiFi à l'IUT slug: >- le-wifi-a-l-iut ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Le réseaux WiFi "eduroam" est accessible, grâce à vos identifiants, dans toutes les universités participants au projet.

Les avantages du réseaux "eduroam" :

  • connexion sécurisée
  • connexion automatique
  • accès au WiFi "eduroam" dans les universités en France et à l'étranger
notes: >-

Plus d'informations : www.eduroam.fr - lien externe

alt: >- credit: >- - kind: block template: title title: >- Paramétrer son appareil slug: >- parametrer-son-appareil ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: alt_background text: >-

Pour vous identifier sur le réseau eduroam, vous devez ajouter le domaine de votre établissement à votre identifiant.
Par exemple : pour le compte informatique jdupond002, l'identifiant eduroam sera jdupond002@u‑bordeaux‑montaigne.fr

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Sur Windows slug: >- sur-windows ranks: self: 3 data: layout: no_background text: >-
  1. Sélectionnez eduroam dans les réseaux proposés
  2. Entrez vos identifiants de l'université (avec le domaine @u-bordeaux-montaigne)
notes: >-

Pour supprimer la connexion : faites un clic droit sur "eduroam" dans la liste des réseaux wifi, puis cliquez sur "Oublier".

alt: >- credit: >- - kind: block template: chapter title: >- Sur Mac slug: >- sur-mac ranks: self: 3 data: layout: no_background text: >-
  1. Téléchargez le fichier de configuration : profil eduroam - lien externe
  2. Cliquez sur le profil eduroam pour l'exécuter
  3. Suivez les instruction pour installer le profil
  4. Entrez vos identifiants de l'université (avec le domaine @u-bordeaux-montaigne.fr)
notes: >-

Pour supprimer la connexion : Ouvrez les réglages, dans le champ de recherche tapez "Profils", puis sélectionnez "eduroam" et cliquez sur le bouton " - ".

alt: >- credit: >- - kind: block template: chapter title: >- Sur mobiles slug: >- sur-mobiles ranks: self: 3 data: layout: no_background text: >-
  1. Téléchargez l'application "geteduroam" sur votre appareil mobile :
    • Google Play Store - lien externe
    • Apple App Store - lien externe
  2. Sélectionnez votre établissement Université Bordeaux Montaigne en tapant le nom dans le champ de recherche
  3. Cliquez sur "Connecter"
  4. Entrez vos identifiants de l'université (avec le domaine @u-bordeaux-montaigne.fr)
notes: >-

Pour supprimer la connexion : supprimez simplement l'application "geteduroam" de votre appareil.
