L'IUT dispose de plusieurs bornes d'accès WiFi qui permettent d'accéder à internet depuis un ordinateur personnel ou appareil mobile.

body: >-

Le WiFi à l'IUT
Mise à jour sur le Wi-Fi Eduroam
Un changement de certificat affecte notre connexion au réseau Wi-Fi eduroam depuis jeudi 25 septembre à 8 h 30.  En fonction de votre configuration actuelle, un message vous invitera à confirmer votre connexion au réseau eduroam. Sinon, vous devez réinitialiser votre paramétrage. Pour cela, supprimez votre configuration eduroam et connectez-vous de nouveau en suivant la procédure ci-dessous selon votre appareil . 


Le réseaux WiFi " eduroam " est accessible, grâce à vos identifiants, dans toutes les universités participants au projet. Les avantages du réseaux "eduroam" :
connexion sécurisée
connexion automatique
accès au WiFi "eduroam" dans les universités en France et à l'étranger

Plus d'informations : www.eduroam.fr

Paramétrer son appareil

Pour vous identifier sur le réseau eduroam, vous devez ajouter le domaine de votre établissement à votre identifiant. Par exemple : pour le compte informatique jdupond002, l'identifiant eduroam sera jdupond002 @u ‑ bordeaux ‑ montaigne.fr

Sur Windows
Sélectionnez eduroam dans les réseaux proposés
Entrez vos identifiants de l'université (avec le domaine @u-bordeaux-montaigne)

Pour supprimer la connexion :
faites un clic droit sur "eduroam" dans la liste des réseaux wifi,
puis cliquez sur "Oublier".

Sur Mac
Téléchargez le fichier de configuration : profil eduroam

Cliquez sur le profil eduroam pour l'exécuter
Suivez les instruction pour installer le profil
Entrez vos identifiants de l'université (avec le domaine @u-bordeaux-montaigne.fr)

Pour supprimer la connexion :
ouvrez les réglages,
dans le champ de recherche, tapez "Profils", puis cliquez sur "Gestion de l'appareil"
sélectionnez le profil "eduroam" et cliquez sur le bouton " - ", puis confirmez en cliquant sur "Supprimer".

Sur mobiles
Téléchargez l'application "geteduroam" sur votre appareil mobile :

Google Play Store
Apple App Store


Sélectionnez votre établissement Université Bordeaux Montaigne en tapant le nom dans le champ de recherche
Cliquez sur "Connecter"
Entrez vos identifiants de l'université (avec le domaine @u-bordeaux-montaigne.fr)

Pour supprimer la connexion : supprimez simplement l'application "geteduroam" de votre appareil.
