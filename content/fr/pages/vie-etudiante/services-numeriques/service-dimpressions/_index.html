Vous pouvez lancer une impression depuis n'importe quel ordinateur de l'IUT puis confirmer l'édition du document sur une des 4 imprimantes. Vous êtes facturé uniquement après avoir confirmé l'impression sur le copieur.
Pour vous identifier sur les copieurs multifonctions il suffit de placer votre carte Aquipass sur le lecteur de carte (vous devrez associer votre carte à votre compte informatique personnel lors de la première utilisation).
Une fois identifié, vous pouvez imprimer vos documents en attente ou faire des photocopies. L'écran de connexion vous affiche en temps réel le solde de votre porte-monnaie électronique. Lorsque vous avez terminé d'utiliser l'imprimante, pensez à vous déconnecter en appuyant sur le bouton rouge sur l'écran de connexion.

notes: >- alt: >- credit: >- - kind: heading title: >- Le porte-monnaie électronique IZLY slug: "le-porte-monnaie-electronique-izly" position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Le système d'impression utilise votre prote-monnaie électronique IZLY pour payer les impressions et photocopies. Le compte IZLY est lié à votre carte d'étudiant Aquipass, il est donc strictement personnel.
Vous pouvez recharger votre compte en ligne sur le site izly.fr, via une application mobile ou dans certains services du CROUS, notamment les restaurants universitaires. L'IUT ne peut pas encaisser votre argent pour recharger votre compte IZLY.
Afin de pouvoir utiliser votre compte IZLY vous devez au préalable l'activer en suivant la procédure qui vous a été envoyé sur votre boite e-mail étudiant.

IZLY est un service proposé par le CROUS : Votre carte Aquipass IZLY
